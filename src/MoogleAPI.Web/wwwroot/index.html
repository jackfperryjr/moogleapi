Endpoints

Characters
GET
/api/characters
List all characters · query params: gameId, page, pageSize
GET
/api/characters/{id}
Get a single character by ID
GET
/api/characters/search
Search by name or description · query params: query, gameId
Monsters
GET
/api/monsters
List all monsters · query params: gameId, category, page, pageSize
GET
/api/monsters/{id}
Get a single monster by ID
GET
/api/monsters/search
Search by name or description · query params: query, gameId, category
Games
GET
/api/games
List all games · query params: page, pageSize
GET
/api/games/{id}
Get a single game by ID, including character and monster counts

Quick Example

# Search for characters named Cloud
GET https://api.moogleapi.com/api/characters/search?query=Cloud
─── Response ───────────────────────────────────────
{ "results": [ { "id": 1, "name": "Cloud Strife", "role": "Protagonist", "description": "A former SOLDIER turned mercenary...", "gameName": "Final Fantasy VII" } ] }
